# psicólogo clínico en CDMX🌐💻

¡Bienvenido a psicólogo clínico en CDMX!  
Este proyecto está diseñado con **HTML** y **CSS puro**, enfocado en ser **moderno, minimalista y responsivo**. Ideal para mostrar tus habilidades como desarrollador, diseñador o profesional independiente.

## Características ✨

- Página HTML + CSS completamente responsiva 📱
- Tipografías modernas con Google Fonts ✒️
- Paleta de colores profesional y adaptable 🎨
- Estilo limpio, minimalista y elegante 🧼
- Incluye botón de agendamiento con **Calendly** 📅
